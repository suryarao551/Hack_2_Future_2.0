Three-Model Intelligence Stack

Each model is the best-in-class for its task, benchmarked against multiple alternatives on the DISPLACE dataset.

DIARIZATION
Pyannote 3.1
End-to-end neural diarization pipeline. Handles overlapping speech natively via segmentation-3.0. Tested against 7 models — lowest DER by a wide margin.

IndicConformer
600M parameter multilingual conformer built for Indic languages. Runs in FP16 with CTC decoding. Zero hallucination loops on dialect-heavy Hindi — unlike SeamlessM4T.

7B instruction-tuned LLM running locally with 4-bit NF4 quantization. Temperature 0.3 ensures deterministic clinical output. Full pipeline runs on a single GPU.
